# 简介

## 什么是云监控服务（CES）

云监控服务（Cloud Eye Service，CES）是华为云提供的监控和告警服务，用于实时监控云资源的运行状态和性能指标。CES服务支持多种云资源的监控，包括弹性云服务器、弹性负载均衡、关系型数据库、对象存储服务等，提供丰富的监控指标、灵活的告警规则配置和多种通知方式，帮助用户及时发现和处理资源异常，保障业务稳定运行。

CES服务提供实时监控、历史数据查询、告警管理、自定义监控等功能，支持多种告警通知方式，包括邮件、短信、HTTP/HTTPS回调等，具备高可用性和高可靠性。通过CES服务，企业可以全面掌握云资源的运行状态，实现主动运维和智能告警，提升业务可用性和用户体验。
