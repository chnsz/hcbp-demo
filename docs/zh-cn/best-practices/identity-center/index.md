# 简介

## 什么是身份中心（Identity Center）

身份中心（Identity Center）是华为云提供的统一身份管理服务，支持跨云、跨应用的统一身份认证和授权管理。身份中心提供集中化的用户管理、用户组管理、权限策略配置等功能，帮助企业实现统一的身份管理和访问控制，满足企业级身份管理和合规要求。

身份中心服务支持多种身份源集成，包括本地目录、外部身份提供商等，提供单点登录（SSO）、多因素认证（MFA）等安全功能。通过身份中心服务，企业可以实现跨云、跨应用的统一身份管理，提高身份管理的效率和安全性，保障业务系统的安全稳定运行。
