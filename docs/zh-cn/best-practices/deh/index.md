# 简介

## 什么是专属主机（DEH）

专属主机（Dedicated Host，DEH）是华为云提供的物理服务器资源，用于满足对资源独享、安全合规等有特殊要求的业务场景。专属主机提供物理服务器的完全控制权，实现资源的物理隔离，满足合规性要求。通过专属主机，企业可以在专属的物理服务器上部署ECS实例，获得更高的安全性和资源隔离性。

专属主机服务提供多种主机类型选择、灵活的计费方式、自动放置功能等，支持包年包月和按需计费，具备高可用性和高可靠性。通过专属主机服务，企业可以满足对资源独享、安全合规等有特殊要求的业务场景，实现资源的物理隔离和完全控制。
