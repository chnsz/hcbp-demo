# 简介

## 什么是云连接（CC）

云连接（Cloud Connect，CC）是华为云提供的全球网络连接服务，支持多种网络实例接入，帮助客户构建全球一张网。云连接服务提供全网互联能力，实现云上网络任意两点互联，保证网络转发一跳可达，无须中转。通过云连接，企业可以简单、快速地构建跨区域、跨网络的统一网络架构。

云连接服务提供简单灵活的配置方式，只需三步即可分钟级构建跨区域跨境多VPC互通网络，支持混合云架构。云连接依托华为全球网络基础设施，提供低时延、高质量的网络体验，同时提供全球一站式合规的网络能力，支持用户专注自身业务创新。
