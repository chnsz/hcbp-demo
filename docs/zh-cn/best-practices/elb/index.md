# 简介

## 什么是弹性负载均衡（ELB）

弹性负载均衡（Elastic Load Balance，ELB）是一种将访问流量自动分发到多台云服务器的服务，能够扩展应用系统对外的服务能力，提高应用的可用性。ELB通过健康检查自动隔离异常的后端服务器，确保业务的高可用性。华为云ELB支持多种负载均衡算法，包括轮询、加权轮询、最少连接等，可满足不同业务场景的需求。

弹性负载均衡提供专享版和共享版两种类型，支持TCP、UDP、HTTP、HTTPS等多种协议，可与ECS、VPC、安全组等服务无缝集成。通过ELB，您可以实现流量分发、会话保持、健康检查等功能，为应用系统提供高可用、高性能的负载均衡服务。
