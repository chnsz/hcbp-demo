# 简介

## 什么是裸金属服务器（BMS）

裸金属服务器（Bare Metal Server，BMS）是华为云提供的专属物理服务器，提供安全可靠的专属计算资源，无虚拟化开销，满足高性能计算、数据库、大数据分析等对性能要求较高的业务场景。BMS提供物理服务器的完整控制权，支持自定义操作系统、网络配置和安全策略，适用于对性能、安全性和合规性有严格要求的应用场景。

BMS服务支持多种实例类型，提供不同的计算能力、存储空间和网络性能，满足不同业务需求。BMS支持VPC和安全组隔离，提供安全可靠的专属计算资源，满足企业对数据安全和监管的需求。通过BMS，企业可以获得物理服务器的完整性能和隔离性，同时享受云计算的灵活性和便捷性。
